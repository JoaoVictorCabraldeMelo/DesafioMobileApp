# DesafioMobileApp

Como utilizar o projeto

```bash
  cd DesafioMobileApp
```

## Caso não tenha o expo na máquina 

```bash
  npm i -g expo-cli
```

## Instale os pacotes antes de executar o projeto 

```bash
  npm install
```

## E então startar o projeto

```bash
  expo start
```

## Aparecera escolhas entre emuladores no local da web
- Web
- Simulador IOS 
- Emulador Android
